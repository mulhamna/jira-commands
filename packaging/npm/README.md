# @mulham28/jirac npm package
[![OpenSSF Best Practices](https://www.bestpractices.dev/projects/12742/badge)](https://www.bestpractices.dev/projects/12742)

Install `jirac` from GitHub Releases using npm.

```bash
npm install -g @mulham28/jirac
jirac --help
```

This package downloads the matching prebuilt `jirac` binary for your platform during `postinstall`.

Supported targets:
- macOS arm64
- macOS x64
- Linux x64
- Linux arm64
- Windows x64

Source repo: <https://github.com/mulhamna/jira-commands>
